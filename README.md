# ❤️ Kalbimde

Kalbimde, insanların birbirleriyle tanışıp sohbet edebileceği modern bir flört ve sohbet platformudur.  
Frontend Angular ile geliştirilmiştir, backend ise C#/.NET tabanlıdır. Kullanıcı dostu arayüzü ve ölçeklenebilir mimarisiyle sosyal etkileşimleri kolaylaştırmayı hedefler.

## 🚀 Başlangıç
Yerel geliştirme sunucusunu başlatmak için:
    ng serve
Tarayıcıdan http://localhost:4200/ adresine gidin. Kaynak dosyalarda yaptığınız değişiklikler otomatik olarak yansıyacaktır.

## 🛠️ Kod Üretimi
Yeni bir bileşen oluşturmak için:
    ng generate component component-name
Tüm şematiklerin listesini görmek için:
    ng generate --help

## 📦 Build
Projeyi derlemek için:
    ng build
Derleme çıktıları dist/ klasöründe saklanır. Production build varsayılan olarak performans için optimize edilmiştir.

## ✅ Testler
Unit Test (Vitest):
    ng test
End-to-End Test:
    ng e2e
Angular CLI varsayılan olarak e2e framework içermez, ihtiyacınıza uygun bir framework seçebilirsiniz.
